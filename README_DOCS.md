## ⚡ 3秒快速用法

### CLI 模式（处理本地文件）
```bash
docker-compose --profile cli run analyzer-cli
```

### API 模式（启动服务）
```bash
docker-compose --profile api up -d
# 访问 http://localhost:8000/docs
```
